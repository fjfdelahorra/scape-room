Escape Room

Bienvenido al juego. Resuelve los cuatro puzzles para obtener los tokens y escapar.

¿Cómo levantar el servidor?

  1. Descarga el proyecto: 
    git clone https://github.com/fjfdelahorra/scape-room
  2. Compila el proyecto: 
    mvn package
  3. Ejecuta el servidor HTTP embebido: 
    java --add-modules jdk.httpserver -jar target/escape-room-java-1.0.0.jar

El servicio quedará disponible en http://localhost:8080.

Si prefieres ejecutar sin empaquetar, usa:

mvn exec:java -Dexec.mainClass=com.chapterescape.Main

También puedes ejecutar el proyecto desde IntelliJ usando el botón Run.
